## 작성 규칙

### Commit 메시지를 작성시 아래와 같이 규칙을 정한다.

#### [공부 섹션][행동] 메시지

- ** 필수 ** [공부 섹션] 제일 처음에 해줍니다. 만약에 js 를 공부했다면 [js]
- [add] : 새로 파일 생성 시
- [update] : 파일 갱신
- [migration] : 마이그레이션
